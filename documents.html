1. Introduction & Overview

CC App is a comprehensive web-based utility application designed to provide:

  • Fast unit conversions across multiple categories
  • Advanced mathematical calculations
  • Gaming-specific calculators (DPS, TTK, reload optimization)
  • Function graphing with auto-scaling
  • Dark/Light theme support
  • Responsive design for desktop and tablet use

Technology Stack

  • Backend: Python 3.x + Flask
  • Frontend: HTML5 + CSS3 + Vanilla JavaScript
  • Data: localStorage API for client-side persistence
  • Analytics: Google Analytics (GA4)
  • Deployment: Compatible with any Python WSGI server

2. Current Features

2.1 Standard Unit Converter

Categories Supported:

  • Length: meter, kilometer, centimeter, millimeter, inch, foot, yard, mile
  • Mass: kilogram, gram, milligram, metric ton, ounce, pound, stone
  • Time: second, minute, hour, day, week, year
  • Temperature: Celsius, Fahrenheit, Kelvin
  • Area: square meter, hectare, acre, square km, square foot, square inch
  • Volume: cubic meter, liter, milliliter, gallon, pint, quart, cup, barrel
  • Speed: m/s, km/h, mph, knot, ft/s
API:
POST /api/convert
POST /api/convert-all
Payload: { category, from_unit, value }
Response: { success, result | results }

2.2 Miscellaneous Converter

Categories: Angle, Sound Intensity, Power, Pressure, Data, Frequency, RPM, Firearm ROF

API:
POST /api/convert-misc
Payload: { category, from_unit, value }
Response: { success, results }

Special Handling: Logarithmic dB, π-precision radian, RPM linear/angular conversions.

2.3 Calculator

  • Basic arithmetic: +, -, *, /
POST /api/calculate
Payload: { expr }
Response: { success, result }

2.4 Graphing Calculator

  • 500-point sampling, auto / manual Y-scale
  • Grid, responsive canvas, validation

Functions: sin, cos, tan, asin, acos, atan, log, ln, exp, sqrt, pow, abs, floor, ceil, round, pi, e.

POST /api/graph-sample
Payload: { expr, xmin, xmax, samples }
Response: { success, xs, ys }

2.5 Game Calculator

Tools: RPM & ROF, Reload Time, Weapon Swap, DPS, TTK.

2.6 Settings & Preferences

Theme, default category, decimals, feature toggles, graph resolution, grid toggle (stored in localStorage).

2.7 User Interface

Glassmorphism, gradients, responsive grids, accessibility focus.

Breakpoints: mobile <768px, tablet 768–1024px, desktop >1024px.

3. Architecture

3.1 Frontend Structure

File: static/script.js

  • Navigation: showPage(), helpers
  • Settings: applyTheme(), resetSettings()
  • Converters: updateUnits(), convertUnits(), convertMisc()
  • Graphing: graphPlot(), _graph_sample(), _graph_autoscale(), _graph_draw()
  • Game: multiple calculator methods
